 -------------------------------------------------------------------------------
    ABOUT 
 -------------------------------------------------------------------------------
Code4Net.SlideShow.js is a jQuery plug-in created in order to simplify creation 
of slide show galleries for web and it is developed upon jQuery 1.6.2 version. 
Code4Net.SlideShow has three transition effects between slides (horizontal slide
/vertical slide/fade in - fade out) and can render two types of navigation 
(bullet and arrow style) or can work with no navigation as automatic slide show. 
For more options and how to use this plug-in, please refer to 
code4net.slideshow.documentation.txt

 -------------------------------------------------------------------------------
    RELEASE NOTES
 -------------------------------------------------------------------------------
Code4Net.SlideShow.js v1.0:
  - Three transition effects (horizontal slide/ vertical slide/ fade in - fade out)
  - Automatic slide show(optional)
  - Navigation(optional) two types (bullet/arrow)
  - Slide can be whatever you like (img/div...)
  - Settable slider speed
  - Settable pause between slides
  - Settable pause period (when user clicks on navigation button)
  - Should work with slides of variable dimensions but for best effect you 
    should use slide containers of the same dimensions

 -------------------------------------------------------------------------------
    FUTURE RELEASES
